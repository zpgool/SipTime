# SipTime

하루 동안 마신 물의 양을 간편하게 기록하고 목표 섭취량을 관리할 수 있는 Flutter 기반 미니 프로젝트입니다.

## 프로젝트 소개

SipTime은 사용자가 물을 마실 때마다 기록을 추가하고, 오늘의 총 섭취량과 목표 달성 여부를 확인할 수 있는 간단한 물 마시기 관리 앱입니다.

## 주요 기능

### 홈 화면

* 오늘 마신 물 총량 확인
* `+100mL` 버튼으로 빠르게 추가
* 직접 입력하여 물 섭취량 기록
* 기록 화면 및 설정 화면 이동

### 기록 화면

* 물 마신 기록 리스트 조회
* 개별 기록 삭제
* 전체 기록 초기화

### 설정 화면

* 하루 목표 물 섭취량 설정 및 변경

## 기술 스택

* Flutter
* Dart
